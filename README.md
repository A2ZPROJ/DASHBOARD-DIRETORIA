# Dashboard Diretoria — 2S Engenharia / A2Z Projetos

Dashboard público estático de acompanhamento de serviços (CT-027.2025 — Projetos ACCIONA).

Os dados são lidos automaticamente da planilha interna pelo app Nexus (CodePro) e publicados no Supabase. Esta página consulta a tabela `dashboard_data` via REST anon read e atualiza automaticamente a cada 60 segundos.
